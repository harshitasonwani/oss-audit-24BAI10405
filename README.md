## Project Overview
This project focuses on basic concepts of open source software using shell scripting.  
I worked on different scripts to understand how commands work and how tasks can be automated in a simple way.  
The idea was to learn by doing and get a practical understanding of open source tools.

## Scripts Included

Script 1: System Identity  
This script shows basic system details like username, system name, and date.  
It helped me understand how to access system information using shell commands.

Script 2: Package Inspector  
This script checks whether some common packages are installed or not.  
Since I ran it on Windows using Git Bash, it shows that the package manager is not found.  
On a Linux system, it would work properly.

Script 3: Disk and Permission Auditor  
This script checks different directories and displays their permissions and size.  
It helps in understanding how file access and storage work in a system.

Script 4: Log File Analyzer  
This script searches for a keyword in a file and counts how many times it appears.  
It also shows the last few matching lines.  
Since Linux log files are not available on Windows, I tested it using a sample text file.

Script 5: Manifesto Generator  
This script takes input from the user and creates a text file based on that input.  
It shows how scripts can interact with users and generate output files.
